Part 1 - What This Version Is

The current Hook repo is a successful hackathon MVP. It has a shared dashboard, parallel API calls, deterministic local datasets, and a clear product story. What it does not yet have is durable identity, imported UT data, or production-backed user state.

This version is the bridge from demo to usable product. The objective is to preserve the current polished flow while replacing local-only assumptions with persistence, source-backed imports, structured parsers, and explicit operational fallbacks.

AreaCurrent demo behaviorTarget product behavior
IdentitySession-backed browser profileSupabase-backed account, verified UT email, required UT EID
OnboardingManual courses, GPA range, text resume summaryTranscript upload, resume upload, parsed skills, numeric GPA, optional photo
AcademicCurated JSON degree and course dataDegree-plan normalization, next-term schedule ingestion, course detail pages
OpportunitiesCurated lists and local matchersScheduled imports from UT and external sources with persistent save/apply state
ChatMarkdown-grounded campus support plus optional model outputOfficial-link-first resource registry with grounded citations
The operating principle Keep the current product shape intact. Productization is about making the existing surfaces real, durable, and auditable, not replacing them with a different concept.

Part 2 - Scope Lock

BucketLocked scope
In ScopeSupabase auth, profile persistence, transcript and resume upload, majors import, scheduled ETL snapshots, source-linked recommendations, course and lab detail pages, internship save/applied state, Playwright validation
Allowed FallbacksManual correction after parsing, reviewed import snapshots when sources are brittle, deterministic fallback responses in chat
DeferredOfficial UT SSO, live request-time scraping, mobile app, registrar-grade transcript sync, admin back office
Locked defaults Use Supabase for auth, database, and storage. Refresh data through scheduled ETL snapshots, not live request-time scraping. Limit the student scope to UT Austin undergraduate majors. Prefer direct official UT links in chat whenever a reliable source exists.
